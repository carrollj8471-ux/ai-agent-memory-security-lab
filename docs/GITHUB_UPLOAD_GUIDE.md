# GitHub Upload Guide

## 1. Create a new GitHub repository

Name it:

```text
ai-agent-memory-security-lab
```

Recommended description:

```text
Defensive AI security lab demonstrating memory poisoning risks, detection, and remediation for AI agents.
```

## 2. Initialize Git locally

```bash
git init
git add .
git commit -m "Initial commit: AI Agent Memory Security Lab"
```

## 3. Connect to GitHub

Replace `YOUR_USERNAME`:

```bash
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ai-agent-memory-security-lab.git
git push -u origin main
```

## 4. Add screenshots

Save screenshots into:

```text
screenshots/
```

Then commit:

```bash
git add screenshots/
git commit -m "Add lab screenshots"
git push
```

## 5. Pin the repository

On GitHub, pin the repository to your profile so employers can find it quickly.
