# Security Policy

## Purpose

This repository is a defensive AI security lab. It demonstrates memory poisoning concepts with local mock data and safe simulated payloads only.

## Supported Use

Use this project for:

- Portfolio demonstrations
- Defensive security education
- AI-agent threat modeling practice
- Local detection and remediation exercises
- Testing systems you own or have permission to assess

Do not use this project to target third-party systems, collect credentials, evade monitoring, deploy malware, or perform unauthorized activity.

## Reporting Issues

If you find a security issue in the lab implementation, open a GitHub issue with:

- A short summary
- Affected file or component
- Steps to reproduce
- Expected and actual behavior
- Suggested mitigation, if known

Do not include real secrets, customer data, or third-party sensitive information in issues.
